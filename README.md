# JSON-CSV-Spreadsheet



Le projet consiste en 3 parties :
​
1. CSV_scrapper
2. JSON_scrapper
3. Spreadsheet_scrapper
​
​
​
![](header.png)
​
## Installation
​
OS X & Linux:
​
```sh
bundle install
```

Pour faire tourner les programmes, lancer individuellement dans lib/app/ :
- maire.rb	: json
- csv_mairie.rb : CSV
- Spreadsheet.rb : google drive
